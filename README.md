### Ejecución
#### El Servidor
Primero, correr el servidor

    $ python main.py

Después, poner un mensaje

    $ >>> Hello World

El servidor codificara el mensaje y lo pondrá en un socket. 

#### El Cliente
Correr el cliente
    
    $ python Receptor.py

El cliente tratará de detectar el mensaje y decodificarlo.
